Welcome to Jellycat Collector

This site is a Jellycat wishlist and collection hub created as a student project by Supa-anan Fuchs for WDD 231. It demonstrates semantic HTML, responsive layouts, data-driven JavaScript, localStorage, and accessibility.

Use the Collection page to browse a curated list of Jellycat plushies, the Wishlist page to practice saving your own ideas, and the Tips page for simple care and display inspiration.

Explore the Collection

How This Site Is Organized

Collection (Data & Modal)

Fetches Jellycat data from a JSON file using the Fetch API with async/await and try...catch. Renders at least 15 items dynamically and includes an accessible modal dialog for details.

Wishlist (LocalStorage)

Uses localStorage to save a simple text-based wishlist for this browser. You can add and remove items and see them persist across page reloads.

Tips & Form

Shares quick care and display ideas and includes an HTML form that submits to a separate action page to display submitted data.

Project Video

This project includes a short walkthrough video demonstrating the JavaScript requirements. You can find the link in the footer of each page.
